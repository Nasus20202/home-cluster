# Home Cluster

Localhost home cluster setup using Kubernetes.

## Setup

1. Run `./setup.sh`
2. Run `sudo cat hosts >> /etc/hosts`
3. Run `./trust-ca.sh`

| -------- | ------------ | ---------------------------------------- |
